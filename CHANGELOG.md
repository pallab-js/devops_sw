# Changelog

All notable changes to DevForge will be documented in this file.

## [1.0.0] — TBD

### Added
- Process Manager: create, start, stop, and monitor local processes
- Environment & Secrets Vault: manage .env files with Keychain-backed secrets
- Docker Console: manage containers and images via Docker socket API
- Git Workspace Manager: multi-repo dashboard with full git operations
- SSH & Host Manager: parse SSH config, test connectivity, launch terminals
- Task Runner: discover and run Makefile/package.json/Justfile tasks
- Log Aggregator: tail and search log files with regex filtering
- System Health Dashboard: real-time CPU, memory, disk, network monitoring
- Global search (Cmd+K), preferences window, menu bar extra
